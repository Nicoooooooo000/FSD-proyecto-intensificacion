# FSD-proyecto-intensificacion
#  Cloudy – SPA (Etapa 1)

Repositorio inicial del proyecto **Cloudy**, desarrollado para la intensificación 2025 de Funcionamiento de los Sistemas Digitales.  
En esta etapa solo se entrega el diseño, documentación y estructura del proyecto, sin código.


##  Nombre del proyecto
**Cloudy – Single Page Application**


##  Descripción breve
Cloudy es una SPA diseñada con una estética minimalista, clara y moderna.  
Incluye todas las pantallas necesarias para un sistema de registro, login y panel privado.  
En esta etapa se entregan los bocetos, prototipos y documentación del diseño.


##  Objetivo del proyecto
- Diseñar la estructura visual y funcional de una SPA.  
- Definir la identidad visual (colores, tipografía, componentes, UI).  
- Representar las pantallas obligatorias en formato desktop y mobile.  
- Describir el flujo funcional del usuario según los requisitos técnicos.  
- Preparar la estructura inicial del repositorio para las siguientes etapas.
